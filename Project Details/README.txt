                                                             Twitter Data Analysis Program
                                                                Software information
                                                             

Twitter Data Analysis Program is used to collect twitter data and use that data to find sentiment of individual tweets, the number of specific terms in that twitter data, based on tweet sentiment how happy or sad a state is and the top ten hash-tags used in the data. 
 
Twitter Data Analysis Program is to be run on Ubuntu 16.04 and Python 2.7.12. 

In order to make our PC/Laptops adaptable to the requirements, installation of both Ubuntu 16.04 and Python 2.7.12 is required.

In order to dual-boot (keeping Winows and Ubuntu) the following link can be followed: https://www.youtube.com/watch?
v=qNeJvujdB-0&list=LLcrIsDvrksQhLv-z6Am5UlQ&index=6

For python installation: https://www.python.org/downloads/release/python-2712/

Once the installations are successfully completed, Ubuntu terminal is used to run the Twitter Data Analysis Program. 

Getting started on Ubuntu terminal: https://www.linux.com/learn/get-started-ubuntu-1604
